# Class: ServerError

Error thrown when WordPress REST API returns an error

## Example

```typescript
throw new ServerError({
	serverCode: 'rest_forbidden',
	serverMessage: 'Sorry, you are not allowed to do that.',
	status: 403,
	path: '/wpk/v1/things',
	method: 'POST',
});
```
