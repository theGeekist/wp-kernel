# Function: formatClassModifiers()

```ts
function formatClassModifiers(flags): string[];
```

Formats PHP class modifiers into an array of keywords.

## Parameters

### flags

`number`

The bitmask of class modifier flags.

## Returns

`string`[]

An array of string keywords (e.g., ['abstract', 'final']).
