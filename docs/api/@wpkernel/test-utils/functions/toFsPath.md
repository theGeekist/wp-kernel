# Function: toFsPath()

```ts
function toFsPath(workspace, posixPath): string;
```

Converts a POSIX-style path to a file system path relative to a workspace.

## Parameters

### workspace

`string`

The path to the workspace.

### posixPath

`string`

The POSIX-style path to convert.

## Returns

`string`

The file system path.
